## 상태 코드 정의

| 코드 | 설명 |
|------|------|
| `not_contacted` | 아직 문의하지 않음 |
| `contacted` | 문의 메일/폼 발송 완료 |
| `awaiting_reply` | 회신 대기 중 |
| `approved` | 해당 항목 서면 승인 확인 |
| `rejected` | 거절 또는 조건 불일치 확인 |
| `on_hold` | 검토 보류 (비용·조건 등) |

## Due Diligence 항목 설명

| 항목 그룹 | 항목 | 설명 |

| | 박스스코어 | 이닝별 득점·타수·ERA 등 세부 기록 여부 |
| | 선수 검색 | 이름 기반 선수 검색 API 여부 |
| | 선수 시즌 기록 | 타자/투수 시즌 누적 스탯 여부 |
| | 팀 시즌 기록 | 팀 승패·스탯 여부 |
| | 최근 경기 기록 | 선수/팀 최근 n경기 기록 여부 |
| | 세부 스탯 수준 | 기본 / 고급 / Statcast 수준 여부 |
| **기술 조건** | API 호출 제한 | RPM / RPD 한도 |
| | 캐싱 허용 범위 | 서버 캐시 허용 시간·방식 |
| | 데이터 저장/재배포 | DB 저장 후 다수 사용자 제공 허용 범위 |
| **법적 조건** | 데이터 출처 고지 의무 | Provider 브랜드명·로고 표기 의무 여부 |
| | 공식 MLB 표시 금지 | "공식", "Official" 등 표현 사용 제한 여부 |
| **비용** | 월 비용 | 기본 구독 비용 (예상 범위, TBD) |
| | 초과 호출 비용 | Rate limit 초과 시 과금 방식 |
| | Trial/sandbox | 계약 전 테스트 환경 제공 여부 |
