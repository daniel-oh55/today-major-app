## 평가 기준 설명

| 항목 | 설명 |
|------|------|
| 광고 수익화 허용 | 광고형 무료 모바일 앱(free mobile app monetized with ads)에서 사용 가능 여부 |
| 실시간 스코어 | 진행 중 경기의 실시간 점수 제공 여부 |
| 경기 상태 | 경기 상태(진행/종료/예정/연기 등) 제공 여부 |
| 라인업 | 경기 시작 라인업/선발 투수 정보 제공 여부 |
| 박스스코어 | 이닝별 득점, 타수, ERA 등 세부 기록 제공 여부 |
| 선수 시즌 기록 | 타자/투수 시즌 누적 스탯 제공 여부 |
| 팀 시즌 기록 | 팀 승패, 팀 스탯 제공 여부 |
| 선수 검색 | 이름 기반 선수 검색 API 포함 여부 |
| 세부 스탯 수준 | 제공 스탯의 깊이(기본/고급/Statcast 수준 등) |
| 호출 제한 | 일별/분당 요청 한도 |
| 캐싱 허용 | 서버 캐시 사용 허용 범위 |
| 저장/재배포 | DB 저장 및 다수 사용자 제공 허용 범위 |
| 출처 고지 의무 | Provider 브랜드명/로고 표기 의무 여부 |
| 공식 앱처럼 표시 금지 | "공식", "Official", "공식 기록" 등 표현 사용 금지 여부 |
| 앱/PWA 배포 가능 | 앱스토어/플레이스토어 또는 PWA 배포 허용 여부 |
| 월 비용 | 기본 구독 비용 |
| 초과 호출 비용 | Rate limit 초과 시 과금 방식 |

## 계약 전 필수 확인 항목 (공통)

모든 후보 API에 대해 아래 항목을 계약 전 반드시 확인하세요.

1. **free mobile app monetized with ads** — 광고형 무료 앱에서의 사용 허용 여부
2. 캐싱/저장/다수 사용자 동시 제공 허용 범위
3. 데이터 출처 표기 의무 (Provider 로고, 브랜드명 표시 요구 여부)
4. "공식", "Official", "공식 기록" 등 공식 앱처럼 보이는 표현 금지 여부
5. 앱스토어/플레이스토어 배포 또는 PWA 배포 허용 여부
6. MLB Stats API 제외 — 이 프로젝트에서는 절대 사용하지 않습니다
