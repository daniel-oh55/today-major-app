# MVP QA 체크리스트

> **실제 사용자 테스트 전 확인 목록입니다.**
> 현재 모든 데이터는 더미 데이터 기준입니다. 실제 서비스 전 상업 API 계약 및 별도 QA가 필요합니다.

---

## 1. 홈 화면 (`/`)

- [ ] 오늘 날짜(KST)가 헤더에 표시된다.
- [ ] 경기 목록이 카드 형태로 표시된다.
- [ ] 전체/진행 중/예정/종료 필터 탭이 동작한다.
- [ ] 즐겨찾는 팀 경기 섹션이 즐겨찾기 추가 후 표시된다.
- [ ] 경기 카드 탭 시 경기센터(`/games/[gameId]`)로 이동한다.
- [ ] 경기가 없는 필터 탭 선택 시 EmptyState가 표시된다.
- [ ] 데이터 출처/비공식 안내 문구가 하단에 표시된다.
- [ ] "공식", "Official", "공식 기록" 표현이 없다.
- [ ] 광고 placeholder가 정책대로 표시된다 (`home_top_banner`, `home_game_list_native`).
- [ ] 광고 placeholder 외 실제 광고 SDK/스크립트가 없다.

## 2. 선수 검색 (`/players`)

- [ ] 검색창에 포커스 시 키보드가 올라온다.
- [ ] 2자 미만 입력 시 안내 메시지가 표시된다.
- [ ] 유효한 검색어 입력 후 검색 시 결과 목록이 표시된다.
- [ ] 검색 결과가 없을 때 EmptyState가 표시된다.
- [ ] 오류 발생 시 ErrorState와 "다시 시도" 버튼이 표시된다.
- [ ] 검색 결과 카드 탭 시 선수 상세(`/players/[playerId]`)로 이동한다.
- [ ] 검색 결과 5번째 아이템 이후 `search_result_inline` 광고가 표시된다.

## 3. 경기센터 (`/games/[gameId]`)

- [ ] 스코어보드가 표시된다 (원정팀 vs 홈팀, 점수).
- [ ] 이닝 라인스코어 테이블이 표시된다.
- [ ] 라인업 테이블이 표시된다.
- [ ] 박스스코어 테이블이 표시된다.
- [ ] 이벤트 목록이 표시된다.
- [ ] 경기 공유 카드 및 공유 버튼이 표시된다.
- [ ] 하단에 `gamecenter_bottom_banner` 광고 1개만 표시된다.
- [ ] 경기센터에 `player_detail_mid_native`, `home_game_list_native` 등 타 광고가 없다.
- [ ] 문자중계/AI 요약/pitch-by-pitch 관련 기능이나 표현이 없다.
- [ ] 존재하지 않는 gameId 접근 시 NotFound 페이지로 이동한다.

## 4. 선수 상세 (`/players/[playerId]`)

- [ ] 선수 이름, 팀, 포지션, 번호가 표시된다.
- [ ] 타자/투수 시즌 스탯 요약이 표시된다.
- [ ] 최근 경기 기록 테이블이 표시된다.
- [ ] 즐겨찾기 버튼이 동작한다 (추가/제거 토글).
- [ ] 선수 기록 카드 공유 버튼이 동작한다.
- [ ] 선수 사진이 표시되지 않는다.
- [ ] `player_detail_mid_native` 광고가 표시된다.
- [ ] 존재하지 않는 playerId 접근 시 NotFound 페이지로 이동한다.

## 5. 팀 상세 (`/teams/[teamId]`)

- [ ] 팀 이름, 리그/지구, 구장 정보가 표시된다.
- [ ] 시즌 승패 기록이 표시된다.
- [ ] 팀 시즌 스탯 요약이 표시된다.
- [ ] 주요 선수 목록이 표시된다.
- [ ] 로스터 테이블이 표시된다.
- [ ] 최근 경기 목록이 표시된다.
- [ ] 즐겨찾기 버튼이 동작한다 (추가/제거 토글).
- [ ] 팀 정보 공유 버튼이 동작한다.
- [ ] 비공식 안내 문구가 표시된다.
- [ ] 구단 로고가 표시되지 않는다.
- [ ] `team_detail_mid_native` 광고가 표시된다.
- [ ] 존재하지 않는 teamId 접근 시 NotFound 페이지로 이동한다.

## 6. 즐겨찾기 (`/favorites`)

- [ ] 즐겨찾는 팀 목록이 표시된다.
- [ ] 즐겨찾는 선수 목록이 표시된다.
- [ ] 목록이 비어있을 때 EmptyState가 표시된다.
- [ ] 팀 항목 탭 시 팀 상세(`/teams/[teamId]`)로 이동한다.
- [ ] 선수 항목 탭 시 선수 상세(`/players/[playerId]`)로 이동한다.
- [ ] 즐겨찾기 제거 버튼이 동작한다.
- [ ] `favorite_home_inline` 광고가 표시된다.
- [ ] BottomNav "즐겨찾기" 탭이 이 화면에서 활성화된다.

## 7. 공유 기능

- [ ] 선수 기록 카드 공유 텍스트가 생성된다.
- [ ] 팀 정보 공유 텍스트가 생성된다.
- [ ] 경기 공유 텍스트가 생성된다.
- [ ] Web Share API → Clipboard → 수동 복사 fallback이 동작한다.
- [ ] 공유 텍스트에 "공식", "Official", "공식 기록" 표현이 없다.
- [ ] 공유 완료 후 `share_complete_interstitial` 광고가 자동 overlay가 아닌 인라인으로 표시된다.

## 8. 광고 placeholder QA

- [ ] 실제 광고 SDK import가 없다 (`window.adsbygoogle` 등).
- [ ] 광고 ID(Publisher ID, App ID)가 하드코딩되어 있지 않다.
- [ ] 경기센터에 `gamecenter_bottom_banner` 광고 1개만 있다 (중간 네이티브/전면 없음).
- [ ] `share_complete_interstitial`은 공유 완료 후 인라인으로만 표시된다.
- [ ] 홈 광고: `home_top_banner`, `home_game_list_native` 2종만.
- [ ] `favorite_home_inline`은 즐겨찾기 페이지에만 표시된다.

## 9. 데이터 출처/비공식 안내 QA

- [ ] 홈 화면에 비공식 안내가 표시된다.
- [ ] 팀 상세 화면에 비공식 안내가 표시된다.
- [ ] DataSourceNotice에 더미 데이터 기준임이 표시된다.
- [ ] "공식 MLB 데이터", "공식 기록" 표현이 앱 전체에 없다.
- [ ] attributionText가 실제 사용 중인 provider 기준으로 표시된다.

## 10. 모바일 375px QA

- [ ] 홈 화면이 375px에서 가로 스크롤 없이 표시된다.
- [ ] 경기센터 라인스코어/박스스코어 테이블이 `overflow-x-auto`로 처리된다.
- [ ] 검색 input이 손가락으로 누르기 쉬운 크기다 (최소 44px 높이).
- [ ] 즐겨찾기/공유 버튼이 손가락으로 누르기 쉬운 크기다 (최소 44px).
- [ ] 필터 탭 버튼이 손가락으로 누르기 쉽다.
- [ ] BottomNav 항목 탭 영역이 충분하다.
- [ ] 카드 간격이 너무 좁지 않다 (최소 gap-2 이상).
- [ ] 상하 여백이 과도하지 않다.

## 11. 내비게이션/라우팅 QA

- [ ] BottomNav "경기" 탭이 `/`에서 활성화된다.
- [ ] BottomNav "선수" 탭이 `/players`에서 활성화된다.
- [ ] BottomNav "즐겨찾기" 탭이 `/favorites`에서 활성화된다.
- [ ] 경기 카드 탭 → 경기센터 이동 ✓
- [ ] 선수 검색 결과 카드 탭 → 선수 상세 이동 ✓
- [ ] 팀 상세 주요 선수 탭 → 선수 상세 이동 ✓
- [ ] 즐겨찾기 팀 항목 탭 → 팀 상세 이동 ✓
- [ ] 즐겨찾기 선수 항목 탭 → 선수 상세 이동 ✓
- [ ] 잘못된 ID 접근 시 NotFound 페이지(`/`)로 안내 ✓
- [ ] NotFound 페이지에서 홈으로 이동 버튼이 동작한다.

## 12. 권리 리스크 QA

- [ ] MLB/구단 로고 이미지가 표시되지 않는다.
- [ ] 선수 사진이 표시되지 않는다.
- [ ] 영상/하이라이트 요소가 없다.
- [ ] 선수 사진 URL이 코드 어디에도 없다.
- [ ] 구단 로고 URL이 코드 어디에도 없다.
- [ ] "공식 MLB 앱", "Official MLB App" 표현이 없다.
- [ ] 한국어 문자중계/AI 요약/pitch-by-pitch 기능이 없다.

## 13. API Key/보안 QA

- [ ] `NEXT_PUBLIC_` 접두사 변수에 API Key가 없다.
- [ ] API Key가 클라이언트 번들에 포함되지 않는다.
- [ ] `.env.local`이 `.gitignore`에 포함되어 있다.
- [ ] API Key가 코드 어디에도 하드코딩되어 있지 않다.
- [ ] MLB Stats API(api.mlb.com) 호출 코드가 없다.
- [ ] 실제 외부 API 호출 코드가 없다 (DummyProvider만 사용).
- [ ] `server-only` import가 provider/env 파일에 있다.

## 14. 빌드/배포 QA

- [ ] `npm run lint` 오류 없음.
- [ ] `npx tsc --noEmit` 오류 없음.
- [ ] `npm run build` 성공 (12개 라우트 이상).
- [ ] 빌드 결과에 `BASEBALL_API_KEY` 등 secret이 포함되지 않는다.
- [ ] `.env.local`이 빌드 결과에 포함되지 않는다.
