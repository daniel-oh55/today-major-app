# 상업 API 계약 전 체크리스트

> **주의:** 이 문서는 계약 전 확인용 가이드입니다. 각 항목의 허용/제한 여부는 반드시 해당 API 제공사와 **직접 계약 전 확인**이 필요합니다. 아래 내용은 단정하지 않으며 "확인 필요" 상태로만 기록합니다.
>
> **핵심 원칙:** 광고형 무료 앱 사용 가능 여부를 가장 먼저 확인하세요. 이 항목이 승인되지 않으면 나머지 항목은 의미가 없습니다.
>
> **API 계약 전에는 실제 production 데이터 소스로 사용하지 않습니다.** DummyProvider가 기본 데이터 소스입니다.
>
> **MLB Stats API (api.mlb.com)는 정식 데이터 소스로 사용하지 않습니다.** 이 원칙은 변경되지 않습니다.

## 1. 공통 확인 항목

## 3. 내부 원칙 (계약 전 고정 사항)

이 앱이 어떤 API를 선택하더라도 아래 원칙은 변경되지 않습니다.

- API Key는 서버 환경변수(BASEBALL_API_KEY)에만 보관, 클라이언트에 노출 금지
- `NEXT_PUBLIC_` 접두사 변수에 비공개 키 사용 금지
- MLB/구단 로고, 선수 사진, 영상, 하이라이트 사용 금지
- UI에서 "공식", "Official", "공식 기록" 등 공식 앱처럼 보이는 표현 금지
- 계약 확인 전 실제 API를 프로덕션 데이터 소스로 사용 금지
